# Dyslexia Text Adaptation System (LARF)

Данный репозиторий содержит MVP системы адаптации текстов для людей с дислексией, разработанный в рамках ВКР.

## Функциональные возможности
- **Режим A**: Генеративное упрощение текста с использованием модели семейства T5 и последующая визуальная разметка.
- **Режим B**: Сохранение оригинального текста с наложением интеллектуальной разметки сущностей (методология Let AI Read First).

## Технологический стек
- **Frontend**: Streamlit
- **NLP**: Natasha (NER, сегментация), HuggingFace Transformers (T5)
- **Deployment**: Streamlit Cloud (CPU optimized)
